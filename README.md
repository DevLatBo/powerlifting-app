# POWERLIFTING-APP

Web app hecho con react.js version 17 para la practica del deporte **Powerlifting** o llamado tambien **Levantamiento en Potencia**.
En esta aplicacion lo que se busca es guardar los registros personales historicos del usuario y considerar 
tambien sus levantamientos maximos en cada uno de los movimineotos del powerlifting:

* Deadlift
* Squat
* Bench Press

Se toma en cuenta en el registro el numero de repeticiones, el peso total de la barra, y la fecha y tiempo 
del momento que se esta realizando la accion.
